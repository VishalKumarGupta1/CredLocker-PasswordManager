# CredLocker-PasswordManager
CredLocker is a secure password manager built with React and Redux. It allows users to safely store and manage their website URLs, usernames, and passwords. With functionality to add, edit, view, and delete entries, CredLocker ensures that your login credentials are stored securely and easily accessible whenever needed.
